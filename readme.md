# Android Architecture Playbook

A practical guide to Android architecture patterns, design principles, and SOLID, explained with comparisons and real-world perspective.

This repository aims to help developers understand **when**, **why**, and **how** to use different architectural approaches in Android projects.

🌍 Languages:

- [English](docs/en)
- [Türkçe](docs/tr)

## Contents

- Architecture Patterns (MVC, MVP, MVVM, MVI, Clean Architecture)
- Design Principles (SOLID, Separation of Concerns)
- Best Practices
- Anti-Patterns
- Comparisons & Notes

## Who is this for?

- Android developers
- Developers preparing for mid/senior roles
